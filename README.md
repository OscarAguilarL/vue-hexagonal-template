# vue-hexagonal-template-v2

## Project setup
```
yarn install
```

### Set Environment Variables

Use .env.local.example as base.

```bash
cp .env.example .env
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
